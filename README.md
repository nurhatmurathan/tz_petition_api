# Petition Api

Test task with simple db entities. Petition Api

## Prerequisites

-   docker engine

## Installation

-   Copy this repository
-   Run this command

```bash
docker-compose up
# ports 3000, 5432 must be open
```

-   open the swagger page http://localhost:3000/docs
